| Category           | Example Command            | Description           |

| Validation         | `$jsonSchema`              | Enforce data rules    |
| Cursor             | `.limit().sort()`          | Iterate query results |
| Read/Write Concern | `{ w: "majority" }`        | Control data safety   |

| Explain            | `.explain()`               | Analyze query plan    |
