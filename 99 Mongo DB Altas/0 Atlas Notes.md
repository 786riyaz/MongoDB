# MongoDB Atlas

---

## Overview

**MongoDB Atlas** is MongoDB’s **fully managed cloud database service**.
Instead of installing and managing MongoDB yourself on a server, Atlas allows you to **deploy and run MongoDB databases in the cloud** with just a few clicks.

> **MongoDB Atlas = MongoDB in the cloud, managed by MongoDB Inc.**

---

## Key Features

* **Cloud-hosted MongoDB**
  Run MongoDB clusters on **AWS**, **Google Cloud**, or **Microsoft Azure**

* **Automatic scaling**
  Easily scale **storage**, **compute**, and **memory** up or down

* **High availability**
  Built-in **replication** and **automatic failover** ensure resilience to outages

* **Security**
  End-to-end encryption, **VPC peering**, **role-based access control**, **IP whitelisting**, and **auditing**

* **Backup & Restore**
  Continuous backups and **point-in-time recovery**

* **Monitoring**
  Real-time metrics, performance charts, and **automated alerts**

* **Serverless & Dedicated Clusters**
  Choose between **serverless databases** or **fixed-size dedicated clusters**

---

## Why People Use MongoDB Atlas

* No need to manage **infrastructure**, **updates**, or **patching**
* Easy integration with applications using **MongoDB drivers**, APIs, and tools
* **Free tier available (M0 cluster)** — ideal for learning, testing, and small applications
